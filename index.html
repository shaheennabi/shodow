Project Details

Project Name

Project Shadow

Project Description

Project Shadow is an innovative smart dustbin created for the school fest. It utilizes an Arduino UNO board, an Ultrasonic sensor, and a servo motor to automate waste disposal. The Ultrasonic sensor detects the presence of objects, triggering the servo motor to open the dustbin lid automatically, providing a touchless and hygienic experience.

Project Goals

  • Design and build a functional smart dustbin prototype
  • Incorporate an Ultrasonic sensor for automatic lid opening
  • Create a user-friendly interface for monitoring and controlling the dustbin
  • Showcase the benefits of automation and technology in waste management
  • Promote environmental consciousness and sustainable practices

Features

  • Automatic lid opening and closing using the Ultrasonic sensor
  • LED indicators for status and feedback
  • Easy-to-use control panel with buttons for manual operation
  • Capacity monitoring to indicate when the dustbin needs to be emptied
  • Efficient waste disposal mechanism to optimize space utilization

Benefits

  • Hygienic and touchless waste disposal
  • Reduction in the spread of germs and bacteria
  • Promotes environmental sustainability
  • Educational tool to demonstrate the application of technology in everyday life
  • Enhanced user experience and convenience
